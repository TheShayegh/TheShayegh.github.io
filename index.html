My Work Experiences

Persian Address Tracer

  • Tehran, Iran
  • Supervisor: Dr. Mohammad Amin Sadeghi
  • Jul. 2020 - Feb. 2021
  • PAT is an intelligent system for converting Persian text of address to geographical coordinates in Tehran. If you are not aware, it is worth mentioning that due to the special conditions in the Persian language and also the lack of standards in addressing in Iran, this is a difficult issue. As far as we know, the PAT system is the most successful and intelligent system available for this issue. This system consists of three phases:

    1.Tokenization and spelling corrections: Due to the characteristics of the Persian language, this step is one of the most difficult steps in all Persian language processing projects. We implemented this phase using complex heuristic search models. It should be noted that the model implemented in this phase is unsupervised and has no assumptions about the input sentences.

    2.Reading the address: A statistical model is built for this phase that tries to determine the geographical boundaries for the mentioned address. This part is the core of the system and many small and large details have been used to increase its accuracy.

    3.Finding the building: A simple search is performed on the buildings in the neighborhood to find the exact location of the address. The concern of this phase was to find and use all available information sources.

    All parts of this project, from the idea generation stage to the data collection and implementation, are all done by me.
